Super Admin Login Process
Open the browser and go to:
https://dev.mehadedu.com/en/super-admin-login
The Super Admin Login interface will appear.
In the WhatsApp Number section:
Select the country code from the dropdown.
Enter a valid WhatsApp number.
Verify number validation:
Invalid numbers should not be accepted.
Number length should follow the required validation rules.
Enter the correct password in the password field.
Check password validations:
Wrong password should show an error message.
Empty password field should not allow login.
Validation message should appear properly.
Click the Continue button.
Verify login functionality:
User should successfully log in with valid credentials.
Invalid credentials should not allow login.
Proper validation/error messages should appear.
Button loading state should work properly.
Session/token should generate successfully after login.
Forgot Password Process
Open the browser and go to:
https://dev.mehadedu.com/en/super-admin-login
On the login page, click the Forgot Password option located below the password field on the right side.
The Forgot Password interface will appear.
In the WhatsApp Number field:
Select the country code from the dropdown.
Enter the registered WhatsApp number.
Verify validations:
Number should not exceed 12 digits.
Invalid numbers should not be accepted.
Empty field validation should work properly.
Click the Send OTP button.
OTP should be sent successfully to WhatsApp.
Success/error messages should display properly.
Enter the OTP in the verification field.
Verify OTP validation:
Invalid OTP should show an error.
Expired OTP should not work.
Empty OTP field should not allow verification.
Click the Verify button.
Enter a new password in the New Password field.
Password must be unique.
New password should not match the old password.
Password validation rules should work properly.
Enter the same password again in the Confirm Password field.
Confirm password must match the new password.
Mismatch should show an error message.
Click the Reset button.
Verify reset password functionality:
Password should reset successfully.
User should be able to log in using the new password.
Old password should no longer work.
Proper success/error messages should appear.
