# Student Profile Settings Process - MehaEdu

## Steps to Reproduce Profile Settings Flow

### 1. Navigate to Website
- Open the MehaEdu website in your browser.

### 2. Student Login
- Log in as a student using valid credentials.

### 3. Access Profile Menu
- After login, in the header, click on the profile icon (student profile section).
- A sidebar (site menu bar) will open.

### 4. Go to Settings
- In the site menu, select the "Settings" option.

### 5. Edit Profile
- In the settings page, click on "Edit Profile."
- This will open a dashboard showing student details, including:
  - Email
  - Location
  - Phone number
  - Date since user joined (month, year)

### 6. Edit Profile Information
- Click on the "Edit Profile" button to modify details.
- Update fields such as email, phone number, and any other personal information.
- Use the "Display Name" option if you want a nickname shown instead of your full name.

### 7. Form Validations
- Ensure all input fields are validated (e.g., correct email format, phone number, required fields).

### 8. Save Changes
- After editing all details, click the "Save Changes" button.
- The updated profile information will be saved.

### 9. Upload Profile Picture
- After saving changes, click on the profile picture area to upload a new photo.
- Select a picture from your local device.
- Once selected, the profile picture will be updated.

---

## Notes
- Ensure all fields are correctly filled before saving changes.
- Validate that the updated details are reflected immediately on the profile page.
- Ensure the profile picture upload works with correct file formats and size limitations.
