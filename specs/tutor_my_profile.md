# Tutor Profile 

## Step-by-Step Process

1. **Tutor Login**
   - Log in to the platform using your credentials.

2. **Access Profile**
   - In the dashboard, at the bottom corner of the side menu, you will see the tutor's avatar along with the tutor’s name and type.
   - Click on it, and a pop-up will appear with options: "My Profile" and "Logout".

3. **Edit Profile**
   - Click on "My Profile".
   - In the profile section, select the "Edit Profile" option.
   - Here, you can update the publicly displayed fields: first name, last name, display name, mobile number, and email.

4. **Validation**
   - Ensure each field is validated:
     - Country code and phone number validation.
     - Email validation.
   - After validation, click "Save".
   - These changes will be saved immediately without needing admin approval.
