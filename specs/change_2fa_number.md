# Super Admin Login Process

1. Go to the provided website link.
2. Select the appropriate country code from the dropdown.
3. Enter a valid WhatsApp number.
4. Click on the **Send Code** button.
5. A verification OTP/code will be sent to the provided WhatsApp number.
6. Enter the OTP in the verification field.
7. Complete the login process successfully.

---

# Update Super Admin 2FA Number

1. After successful login, open the **Sidebar**.
2. Go to **Account Settings**.
3. Multiple setting cards will be available there.
4. Open the first card: **2FA Number**.

## To Update the Number

1. Select the country code.
2. Enter a new valid WhatsApp number.
3. Click on the **Send Verification Code** button.
4. Enter the received verification code/OTP.
5. After successful verification, the new number will be updated as the Super Admin 2FA number.
6. From the next login, the Super Admin must use the newly updated number for login authentication.
